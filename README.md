# awareness
This repository was made in order to post every vulnerability, scam, malware, etc., found in the source code of any desktop app or website.

# The structure
Every app or website will have its own separate folder in which you will find MD files explaining the vulnerability and/or the evidence code.
Most of them are found by other users; in that case, I will always give credit to the user who found it / documented it at the end of every MD file.

# Why?
Why do I make this?
I make this in order for other users to be aware of these kinds of vulnerabilities, scams, malware, etc.
